⚙️ Firebase Setup Guide Admin Only · #setup
Hidden page. Access via yoursite.html#setup
1
Go to console.firebase.google.com → Add Project → name it vks-timesheet → disable Analytics → Create
2
Authentication → Get Started → Enable Email/Password AND Google. Under Settings → Authorized Domains → add abhisheksarda4888.github.io
3
Firestore Database → Create Database → Test mode → Region: asia-south1 (Mumbai)
4
Project Settings (gear icon) → Your Apps → Add Web App → nickname vks-web → Register → Copy firebaseConfig → paste in this file replacing YOUR_API_KEY etc.
5
Authentication → Users → Add User → admin email + password → copy the UID shown. Then Firestore → Start Collection → ID: users → Doc ID = that UID → add fields: name:"Admin", role:"admin", email:"...", pinSet:false, locked:false, pinAttempts:0
6
Upload this HTML file to your GitHub repo Abhisheksarda4888/vks-nagpur → it goes live at abhisheksarda4888.github.io/vks-nagpur/vks-timesheet.html
7
Import Excel columns: name · email · password · department
